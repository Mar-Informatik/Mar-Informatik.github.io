# 个人主页说明

## 本地运行

直接双击打开 `index.html` 即可预览页面。

如果你想用本地服务打开，可以在当前目录运行：

```powershell
python -m http.server 8000
```

然后访问 `http://localhost:8000`。

## 文件说明

- `index.html`：页面结构与文案
- `styles.css`：页面样式
- `script.js`：导航与滚动动效
- `个人主页内容修改表.xlsx`：后续改文案用的表格
- `generate_xlsx.py`：生成上述 `xlsx` 的脚本

## 后续修改方式

优先修改 `个人主页内容修改表.xlsx` 里的内容，然后把改好的表发给我。
我可以继续帮你把修改同步回网页。
